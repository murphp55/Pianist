# Metronome Click Sound

You need to add a metronome click sound file here named `metronome_click.mp3`.

## Options:

### Option 1: Download a free click sound
- Visit https://freesound.org/search/?q=metronome+click
- Download a short (50-100ms) click sound with CC0 license
- Rename it to `metronome_click.mp3`

### Option 2: Generate programmatically
You can generate a simple beep using online tools or audio software:
- Frequency: 1000 Hz
- Duration: 50-100 ms
- Format: MP3

### Option 3: Use a placeholder
For now, the app will work without audio - the visual metronome indicator will still function. Audio errors are gracefully handled and logged.

Place the file at: `assets/sounds/metronome_click.mp3`
